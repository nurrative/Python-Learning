к любой команде можно дописать --help и выйдет описание как использовать эту команду
cd test  -  переход в папку test(если эта папка есть и она существует)
cd ..  -  переход в родительскую папку
cd .  -   переход в эту же папку(никуда не перейдет)
cd  - переход в домашнюю папку(~)   cd(~)
pwd  - показывает абсолютный путь до текущей папки
mkdir test -  создание папки test
mkdir test/new_dir - создание новой папки new_dir в уже существующей папке test
touch test.txt  -  создание файла test.txt
rm  test.txt -  удаление файла test.txt 
rm -rf  -  удаление test.txt со всем ее содержимым
cat test.txt  -  просмотр содержимого файла
ls  test  -  просмотр всех файлов и папок внутри папки test
ls -a test  - просмотр всех файлов и папок включая скрытые
nano test.txt  -  открыть файл в редакторе nano

      ctrl x
